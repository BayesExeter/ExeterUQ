# README

Please make sure that the following packages are installed on RStudio:

* GenSA
* far
* fields
* lhs
* maps
* mco
* mvtnorm
* ncdf4
* parallel
* rstan
* shape
* tensor
* withr
* loo
* bayesplot

In order to install and load 'lhs' package in RStudio console type:
```
install.packages("lhs")
library("lhs")
```
